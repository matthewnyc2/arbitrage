Every trade, one row each

Each row below is one resolved Polymarket event. The strategy either placed a trade or skipped it. When it traded, you can see exactly what it paid, what it got back, and whether it won money.

Where this data comes from

Every event below was pulled live from Polymarket's public Gamma API (the same one their website uses). These are real resolved markets from the past few months — events that have already concluded, with the winner decided. The “last trade price” for each outcome is the final price Polymarket recorded before the market closed.

The backtest runs entirely in your browser: when you select a strategy, your browser applies the strategy's rule to each event and tallies the result. You can open the developer console and see the math. Nothing is pre-computed, cherry-picked, or faked.
